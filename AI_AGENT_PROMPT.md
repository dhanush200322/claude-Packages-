# AI Agent Instructions

Read the repository first.

Analyze:

* README.md
* package.json
* folder structure
* installed packages
* existing components
* existing architecture

Understand repository standards before making changes.

Then analyze the product requirements.

Create:

1. Architecture Plan
2. Page Structure
3. Component Tree
4. State Management Plan
5. UI Plan

Use only installed packages whenever possible.

Reuse existing components before creating new ones.

Do not start coding until analysis is complete.
