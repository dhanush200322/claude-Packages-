# UI/UX Pro Max — Starter

This scaffold provides a minimal Next.js + Tailwind CSS + Framer Motion starter tailored for UI/UX Pro Max outputs.

Quick start:

1. cd uipro-site
2. npm install
3. npm run dev

Files of interest:
- `pages/index.tsx` — main landing page (includes `components/Hero.tsx`)
- `components/Hero.tsx` — 21st.dev-style hero with Tailwind and Framer Motion
